# Task 2 Completed – Number Guessing Game

I have successfully completed **Task 2** of my APSCHE internship by developing a **Number Guessing Game**.

### Project Overview:

This project is an interactive game where the user tries to guess a randomly generated number within a limited number of attempts. It gives hints based on whether the guessed number is too high or too low.

### Features Implemented:

* Random number generation
* User input handling
* Hint messages (too high / too low)
* Attempt counter
* Win/lose condition
* Replay option
* Simple and user-friendly interface

### Technologies Used:

* Python
* Random module
* Console-based interaction

### Outcome:

This task improved my understanding of logic building, conditional statements, loops, and user input handling in Python.
